Root UI with tailwind

---

## How to run

### Dev

```bash
npm run start:dev
```

### Production

```bash
npm run start
```

### Test

```bash
npm run test
```

#### Lint

```bash
npm run lint
npm run lint:fix
```

#### Prettier

```bash
npm run prettier
```
